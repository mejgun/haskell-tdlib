Constructors

MessageLinkInfo

Contains information about a link to a message or a forum topic in a chat

Fields

  • for_album :: Maybe Bool

    True, if the whole media album to which the message belongs is linked

  • media_timestamp :: Maybe Int

    Timestamp from which the videoaudiovideo note/voice note playing must start, in seconds; 0 if not specified. The media can be in the message content or in its web page preview

  • message :: Maybe Message

    If found, the linked message; may be null

  • message_thread_id :: Maybe Int

    If found, identifier of the message thread in which to open the message, or a forum topic to open if the message is missing

  • chat_id :: Maybe Int

    If found, identifier of the chat to which the link points, 0 otherwise

  • is_public :: Maybe Bool

    True, if the link is a public link for a message or a forum topic in a chat
